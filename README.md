Python 12306 抢票小工具
-----------------
[12306](http://www.12306.cn/)用python实现的12306抢票小工具

python版本支持
-------------

3.5以上

Usage
---------------------------------
1. 安装依赖库, `pip install -r requriement.txt`
2. 进入python12306文件夹,然后复制config.yaml_exmpale为config.yaml,按照说明修改配置文件
3. 安装好依赖之后,进入python12306文件夹, 运行`python run.py`来进行抢票
